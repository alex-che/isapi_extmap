========================================================================
    CONSOLE APPLICATION : tests Project Overview
========================================================================

This is a unit test project, which uses Boost Unit test library 
with the STLPort library. To load and build it, you need to correctly 
set the BOOST and STLPORT macros in the 
local_development_environment.vsprops file. Also check, that the
AdditionalLibraryDirectories setting points to existent and appropriate
directories.
